신청 전 체크리스트

강좌를 공개하기 전에 꼭 먼저 맞춰야 하는 항목입니다.

  • 신청기간이 이번 운영 월과 맞는지 확인
  • 강의시간과 학년 제한이 정확한지 점검
  • 수강료, 대기정원, 강사ID 입력 여부 확인
  • 학생 노출이 필요한 강좌만 출력 상태로 전환

신청 진행 중 확인

접수 중에는 상태보다 실제 신청 흐름을 자주 확인하는 편이 안전합니다.

  • 신청 수와 정원 비율을 보고 증설 여부 판단
  • 대기자 발생 강좌는 대기 정원과 이동 계획 점검
  • 신청자관리에 바로 연결해 누락이나 중복 신청 확인
  • 필요 시 엑셀 출력으로 담당자 공유 자료 생성

마감 전 운영 흐름

강좌 상세를 따로 열지 않아도 이 화면에서 다음 액션을 바로 판단할 수 있게 정리했습니다.

  • 강사 마감과 환불 마감 시점을 강좌별로 분리 관리
  • 상태가 종료인 강좌는 신청 불가 안내 기준으로 사용
  • 출석부 출력 전 신청자 최종 확정 여부 확인
  • 문의가 있으면 매뉴얼과 고객지원 게시판으로 연결
