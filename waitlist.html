우선순위 확인

대기자 화면은 순위가 가장 중요한 판단 기준이라 먼저 정리해 두는 편이 좋습니다.

  • 순위 1~3은 결원 발생 시 가장 먼저 검토
  • 같은 강좌 안에서 학년과 시간 중복 여부도 함께 확인
  • 연락처가 있으면 즉시 안내 대상인지 판단 가능
  • 등록일자는 최근 대기 변동 여부를 볼 때 유용

신청 전환 흐름

신청 버튼은 단순 이동이 아니라 정원, 대기 순위, 기존 신청 상태를 같이 보는 흐름이 안전합니다.

  • 강좌 정원과 현재 신청자 수를 먼저 확인
  • 상위 순위자부터 신청 전환 여부를 검토
  • 신청 전환 후에는 신청자관리 화면에서 반영 상태 점검
  • 필요 시 환불·취소 화면과 함께 교차 확인

삭제 전 체크

대기 삭제도 기록상 영향이 있으므로 연락처와 순위, 강좌명을 함께 보고 판단하는 편이 좋습니다.

  • 동명이인은 학년, 반, 번호까지 같이 확인
  • 보호자 연락처가 있으면 안내 후 삭제 여부를 검토
  • 대기 삭제는 차순위자 영향까지 고려해야 함
